# Profile Photo Placeholder

This folder should contain your professional profile photo.

## Required Image

- **Filename**: `profile.jpg` (or update the path in `index.html`)
- **Recommended Size**: 800x800 pixels (square format)
- **Format**: JPG or PNG
- **Quality**: High resolution for best display

## Tips for a Great Profile Photo

1. **Professional appearance** - Dress appropriately for college applications
2. **Good lighting** - Natural light or professional lighting
3. **Clean background** - Solid color or simple background
4. **Face clearly visible** - No sunglasses, clear facial features
5. **Smiling and approachable** - Friendly, confident expression
6. **Recent photo** - Current representation of yourself

## How to Add Your Photo

1. Save your professional photo in this folder
2. Name it `profile.jpg` OR
3. Update the image path in `index.html`:

```html
<img src="images/YOUR-PHOTO-NAME.jpg" alt="Magda Rajszewski" class="profile-photo">
```

## Additional Photos (Optional)

You can add more photos for a gallery section:
- Activities photos
- Competition photos
- Service project photos
- Performance photos

Name them descriptively:
- `cross-country.jpg`
- `eagle-scout.jpg`
- `musical-performance.jpg`
- etc.

## Image Optimization

Before adding images, consider optimizing them:
- Use [TinyPNG](https://tinypng.com) to reduce file size
- Convert to WebP format for better performance
- Resize to appropriate dimensions (no larger than 1920px)

This keeps your website loading fast!
